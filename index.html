About Topo GPX Viewer

This tool lets you view and explore GPX routes on topographic maps with customizable track styling, distance labels, elevation stats, and more.
The application works on mobile devices, but is best experienced on larger screens.

GitHub Project

Creator: droidgren.github.io

Version: 0.1

Changelog
  • v0.2: Elevation profile bar with interactive cursor tracking. Toggle elevation profile on/off.
  • v0.1: Initial release. GPX route viewer with topographic maps, track styling, distance labels, slope coloring, waypoints, and elevation stats.

This application is fully client-side. No data saved on server.
